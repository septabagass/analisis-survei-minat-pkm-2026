# 📊 Dashboard Analisis Survei Minat PKM 2026

## 📌 Tentang Proyek

Proyek ini bertujuan untuk menganalisis hasil survei minat Program Kreativitas Mahasiswa (PKM) tahun 2026 yang dilakukan oleh PKM Center Universitas Negeri Yogyakarta.

Analisis dilakukan untuk memahami tingkat minat mahasiswa terhadap berbagai skema PKM, mengidentifikasi karakteristik responden, serta menyediakan insight yang dapat digunakan sebagai dasar penyusunan program pembinaan PKM di lingkungan Universitas Negeri Yogyakarta.

---

## 🎯 Tujuan Proyek

- Mengumpulkan dan mengolah data survei minat PKM.
- Membersihkan dan memvalidasi data responden.
- Menganalisis distribusi minat mahasiswa terhadap skema PKM.
- Mengidentifikasi karakteristik dan preferensi peserta.
- Menyajikan hasil analisis dalam bentuk dashboard interaktif.
- Mendukung pengambilan keputusan berbasis data untuk program pembinaan PKM.

---

## 📂 Dataset

Dataset berasal dari hasil survei minat PKM 2026 yang dikumpulkan oleh PKM Center Universitas Negeri Yogyakarta.

### Karakteristik Dataset

- Responden: Mahasiswa Universitas Negeri Yogyakarta
- Sumber data: Google Form
- Format data: Spreadsheet (.xlsx/.csv)
- Periode pengumpulan: Tahun 2026

---

## 🔒 Privasi dan Keamanan Data

Untuk menjaga privasi responden, beberapa informasi pribadi pada dataset telah dianonimkan sebelum dipublikasikan pada repository ini.

Data yang dianonimkan meliputi:

* Nama responden diubah menjadi format `Responden_001`, `Responden_002`, dan seterusnya.
* Nomor induk mahasiswa (NIM) telah disamarkan.
* Nomor telepon diubah menjadi format contoh seperti `08100000001`, `08100000002`, dan seterusnya.
* Alamat email telah disamarkan dan tidak merepresentasikan data asli.

Proses anonimisasi dilakukan untuk melindungi informasi pribadi responden sekaligus memastikan dataset tetap dapat digunakan untuk keperluan analisis dan pembelajaran.

## 🔍 Tahapan Analisis

### 1. Data Collection
Mengumpulkan data hasil survei yang diperoleh melalui Google Form.

### 2. Data Cleaning
Tahapan yang dilakukan:

- Menghapus data duplikat
- Menstandarkan format data
- Memvalidasi jawaban responden

### 3. Exploratory Data Analysis (EDA)

Analisis dilakukan untuk mengetahui:

- Distribusi responden berdasarkan program studi
- Distribusi responden berdasarkan fakultas
- Minat mahasiswa terhadap skema PKM
- Status kepemilikan ide PKM oleh mahasiswa.
- Status kepemilikan tim PKM oleh mahasiswa.

### 4. Data Visualization

Menyusun visualisasi untuk mempermudah interpretasi data dan penyampaian insight kepada stakeholder.

## 💡 Insight Utama

Contoh insight (silakan sesuaikan dengan hasil analisis Anda):

- Mayoritas mahasiswa menunjukkan minat tinggi untuk mengikuti PKM pada tahun 2026.
- Skema PKM-RSH menjadi salah satu skema PKM yang paling diminati oleh mahasiswa.
- Fakultas Matematika dan Ilmu Pengetahuan Alam (FMIPA) merupakan fakultas dengan jumlah peminat PKM terbanyak.
- Mahasiswa angkatan 2024 menunjukkan antusiasme yang tinggi untuk mengikuti Program Kreativitas Mahasiswa (PKM) tahun 2026.
- Dari total 506 mahasiswa yang berminat mengikuti PKM, sebagian besar belum memiliki ide PKM maupun tim PKM, sehingga diperlukan program pendampingan dan fasilitasi pembentukan tim.
- Mahasiswa yang telah memiliki ide PKM cenderung sudah memiliki tim, yang menunjukkan adanya hubungan antara kesiapan ide dan kesiapan pembentukan tim.

---

## 🚀 Rekomendasi

- Menyelenggarakan pelatihan PKM bagi mahasiswa pemula.
- Memperkuat sosialisasi pada setiap bidang PKM.
- Menyediakan pendampingan intensif untuk skema PKM yang memiliki minat tinggi.
- Mengembangkan program mentoring oleh mahasiswa yang telah berpengalaman mengikuti PKM.
